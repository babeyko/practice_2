# Инструмент визуализации графа зависимостей

CLI приложение для анализа зависимостей пакетов и построения визуального представления графов.  
Поддерживает режим анализа реального Cargo проекта на GitHub и режим анализа тестового графа из файла.  
Программа использует итеративный DFS, обнаруживает циклы, строит прямые и обратные зависимости и генерирует PlantUML и PNG диаграммы.

---

# Этап 1. Минимальный прототип с конфигурацией

## Цель этапа  
Создать минимальное CLI приложение с передаваемыми параметрами настройки.

## Реализовано

- Чтение аргументов командной строки
- Поддержка аргументов:
  - p package-name имя анализируемого узла
  - r repo URL репозитория real или путь к файлу test
  - mode real или test
  - out имя PNG файла
  - max-depth глубина анализа
  - branch ветка репозитория
  - reverse-deps режим обратных зависимостей для тестового режима
- Проверка корректности параметров
- Вывод всех принятых параметров при запуске

---

# Этап 2. Сбор данных из Cargo

## Цель этапа  
Получить прямые зависимости пакета из файла Cargo.toml.

## Реализовано

- Загрузка Cargo.toml по URL GitHub через raw.githubusercontent.com
- Парсинг секции dependencies
- Игнорирование других секций и комментариев
- Вывод всех прямых зависимостей пакета

---

# Этап 3. Основные операции над графом

## Цель этапа  
Построить граф зависимостей и выполнить анализ через итеративный DFS.

## Реализовано

- Формат графа узел -> список зависимостей
- Загрузка тестового графа из файла
- Проверка корректности формата узлов
- Итеративный DFS без рекурсии:
  - структура стека узел глубина состояние
  - состояния enter и exit
  - поддержка ограничения max-depth
  - поиск циклов
  - список достижимых узлов
  - список рёбер
- Вывод анализа графа:
  - достижимые узлы
  - рёбра
  - циклы

---

# Этап 4. Обратные зависимости

## Цель этапа  
Определить пакеты которые зависят от выбранного узла.

## Реализовано

- Построение обратного графа
- Режим reverse-deps для тестового режима
- Итеративный DFS по обратному графу
- Вывод зависимых пакетов
- Вывод рёбер и циклов обратного графа

---

# Этап 5. Визуализация PlantUML и PNG

## Цель этапа  
Построить визуальное представление графа и сохранить его в PNG.

## Реализовано

- Формирование списка рёбер из результата DFS
- Генерация файла PlantUML puml
- Вызов PlantUML для создания PNG
- Поддержка Windows Li
